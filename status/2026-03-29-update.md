### Source: KPU Elections (kpu.go.id)
- **What:** Election commission — candidate lists, DPT (voter rolls), real-count results
- **Access:** Web portal, some JSON endpoints during election periods
- **Discovered via:** civic-signal-monitor KPU adapter monitoring candidate disqualifications

### Source: SIMBG Building Permits (simbg.pu.go.id)
- **What:** National building permit system — PBG (Persetujuan Bangunan Gedung) applications
- **Access:** Web portal with public search
- **Discovered via:** civic-signal-monitor SIMBG adapter tracking permit anomalies

### Source: CoreTax DJP (coretaxdjp.pajak.go.id)
- **What:** DJP's new core tax administration system, replacing old e-Filing/e-Billing
- **Access:** Login required (NPWP + password). No public API.
- **Note:** Currently not resolving from AU — may be geo-restricted or in maintenance
- **Discovered via:** civic-signal-monitor CoreTax adapter for NPWP validation

### Source: SATUSEHAT (satusehat.kemkes.go.id)
- **What:** Kemenkes national health interoperability platform built on HL7 FHIR
- **Access:** Portal accessible. API requires registration via satusehat.kemkes.go.id/platform
- **Key data:** Hospital records, patient referrals, health facility registry
- **Discovered via:** Kemenkes data in existing tier-4 monitoring

### Source: BPJPH Halal API (cmsbl.halal.go.id)
- **What:** Backend API for BPJPH halal certification search
- **Access:** POST endpoint `https://cmsbl.halal.go.id/api/search/data_penyelia` — no auth needed
- **Note:** Used by LegalKah (legalkah.id) project for halal verification. Returns 1.98M+ business records
- **Discovered via:** cek-investasi/halalkah.id scraper
